# SuperBands

Two bands: above and below the price (like LuxAlgo Reverse Zones)

## Data Format

```js
[<timestamp>, <high1>, <mid1>, <low1>, <high2>, <mid2>, <low2>]
```

## SuperBands.color1
- **Type:** `color`
- **Default:** `'#d80d3848'`

## SuperBands.color1dark
- **Type:** `color`
- **Default:** `'#d80d3824'`

## SuperBands.color2
- **Type:** `color`
- **Default:** `'#1edbbe33'`

## SuperBands.color2dark
- **Type:** `color`
- **Default:** `'#1edbbe15'`
