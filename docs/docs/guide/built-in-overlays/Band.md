# Band

Bands indicator, e.g. BollingerBands

## Data Format

```js
[<timestamp>, <high>, <mid>, <low>]
```

## Band.color
- **Type:** `Color`
- **Default:** `'#b41d70'`

## Band.backColor
- **Type:** `Color`
- **Default:** `$props.color + '11'`

## Band.lineWidth
- **Type:** `number`
- **Default:** `1`

## Band.showMid
- **Type:** `boolean`
- **Default:** `true`
