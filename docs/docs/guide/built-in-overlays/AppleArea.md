# Area

Area chart, stylized similary to Apple Stocks app. 

## Data Format

```js
[<timestamp>, <value>]
```

## Area.colorUp
- **Type:** `color`
- **Default:** `'#57db64'`

## Area.colorDown
- **Type:** `color`
- **Default:** `'#ff3b2f'`

## Area.colorSelect
- **Type:** `color`
- **Default:** `'#49bef4'`

## Area.colorSelectBack
- **Type:** `color`
- **Default:** `'#00a7f455'`

## Area.colorSelectValue
- **Type:** `color`
- **Default:** `'#3b8fed'`

## Area.colorBaseline
- **Type:** `color`
- **Default:** `'#555555'`

## Area.lineWidth
- **Type:** `number`
- **Default:** `1.5`

## Area.showName
- **Type:** `boolean`
- **Default:** `false`
