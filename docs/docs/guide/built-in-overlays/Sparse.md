# Sparse

Sparse data: points, squares, crosses, triangles

## Data Format

```js
[<timestamp>, <value>, <?direction>]
 <value> :: Price/value
 <?direction> :: Triangle direction: 1 | -1
```

## Sparse.color
- **Type:** `Color`
- **Default:** `'#898989'`

## Sparse.size
- **Type:** `number`
- **Default:** `3`
