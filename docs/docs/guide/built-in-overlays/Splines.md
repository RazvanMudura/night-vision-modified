# Splines

Multiple splines

## Data Format

```js
[<timestamp>, <line1>, <line2>, ...]
```

## Splines.lineWidth
- **Type:** `number`
- **Default:** `1`

## Splines.widths
- **Type:** `Array`
- **Default:** `[]`

## Splines.colors
- **Type:** `Array`
- **Default:** `[]`

## Splines.skipNan
- **Type:** `boolean`
- **Default:** `false`
