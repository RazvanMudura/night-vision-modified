# Volume

Regular volume

## Data Format

```js
[<timestamp>, <volume>, <direction>]
 <direction> :: 1 for green volume, - 1 for red volume
```

## Volume.colorVolUp
- **Type:** `color`
- **Default:** `'#41a37682'`

## Volume.colorVolDw
- **Type:** `color`
- **Default:** `'#de464682'`

## Volume.barsHeight
- **Type:** `number`
- **Default:** `0.15, step: 0.1`

## Volume.currencySymbol
- **Type:** `string`
- **Default:** `'$'`

## Volume.showAvgVolume
- **Type:** `boolean`
- **Default:** `true`

## Volume.avgVolumeSMA
- **Type:** `number`
- **Default:** `20`

## Volume.colorAvgVol
- **Type:** `color`
- **Default:** `'#17e2bb99'`
