# Area

Area chart

## Data Format

```js
[<timestamp>, <value>]
```

## Area.color
- **Type:** `color`
- **Default:** `'#31ce31'`

## Area.lineWidth
- **Type:** `number`
- **Default:** `1.25`

## Area.back1
- **Type:** `color`
- **Default:** `$props.color + '15'`

## Area.back2
- **Type:** `color`
- **Default:** `$props.color + '01'`

## Area.dataIndex
- **Type:** `integer`
- **Default:** `1`
